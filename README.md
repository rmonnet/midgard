# Midgard, an Odin Library

Midgard is meant to be a support library for the Odin language.
It contains algorithms, data structures, and other numerical functions that I found useful to develop in Odin.

Use at your own peril.

## Pre-requisite

You need to install the [Odin compiler](https://odin-lang.org/).

If you are using VSCode, I recommend the Odin Language extension by Daniel Gavin ('danielgavin.ols').

## Algorithms and Data Structure

- Union-Find: Set data structure that determines if there is a path between two nodes (`uf__.odin`)
- Stack: The classic Last In - First Out data structure backed by a linked list
- ArrayStack: The classic Last In - First Out data structure based by a growable Array
